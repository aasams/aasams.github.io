Install the program to computer using the installctl tool.

More can be found out about installctl by running installctl -help

Directions To Install Program

  • Navigate to the computer terminal
  • Create the installation directory /root/program with mkdir
    • type: mkdir /root/program in the terminal
  • Start the installation using the installctl tool:
    • Install program to the target directory /root/program
    • Provide the password secret
    • Do this by entering: installctl -root=/root/program -password=secret
  • If errors occur reference the Troubleshooting section

Troubleshooting:

    Common Installctl Errors
  • Error 100: Password is Missing or Incorrect
  • Error 200: Target Directory is not supplied
  • Error 300: Installer found files already in the installation directory. Include the paramater -overwrite to installctl to attempt to overwrite
  • Error 400: Target directory is corrupted, verify using inspect on the directory
  • Error 500: Target installation directory does not exist, it needs to be created
If error is not listed above or persists, collect the Error Code and message details to Tech Support. If you encounter "Command Not Found", make sure command exists and there are no extraneous spaces or typos.
