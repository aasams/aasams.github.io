Install the program to computer using the installctl tool.

More can be found out about installctl by running installctl -help

Directions To Install Program

  • Create the installation root directory /root/program with mkdir
    • type: mkdir /root/program
  • Start installation using installctl tool:
    • installctl -root=/root/program -password=secret

Troubleshooting:

  • Error 100: Password is Missing or Wrong
  • Error 200: Root Directory is not supplied
  • Error 300: Installer found files already in the root directory. Include the paramater -overwrite to installctl
  • Error 400: Root directory is corrupted, verify using inspect on the root directory
  • Error 500: Root directory does not exist
If error is not listed above or persists, collect the Error Code and message details to Tech Support.
